# vim

## `.vimrc`ファイルを分割して管理する方法

https://qiita.com/mrtry/items/66c432dd79ddcaae6066


## 資材を配置

```shell
ln -s ~/dotfiles/vim/config ~/.vim/
```

## .vimrc

```shell
ln -s ~/dotfiles/vim/.vimrc ~/
```

## jetbrains

```shell
ln -s ~/dotfiles/vim/.ideavimrc ~/
```

## neovim

```shell
ln -s ~/dotfiles/vim/nvim ~/.config/
```

# Neovim 環境構築

```shell
brew install nvim
```

```shell
ln -s ~/dotfiles/vim/nvim ~/.config/
```
